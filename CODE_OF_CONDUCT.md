# Code of Conduct / Código de Conducta

## English

### Our Pledge
We want Syncopation to be a welcoming community. Contributors, maintainers, and players pledge to make participation harassment-free regardless of age, body size, disability, ethnicity, gender identity and expression, level of experience, nationality, personal appearance, race, religion, or sexual identity and orientation.

### Our Standards
Positive examples of behavior:
- Showing empathy and respect for different viewpoints and experiences.
- Giving and gracefully accepting constructive feedback.
- Focusing on what is best for the community and the player experience.
- Using inclusive language in code, documentation, and discussions.

Unacceptable behavior includes:
- Harassment, intimidation, or discriminatory jokes and language.
- Publishing private information without explicit permission.
- Trolling, insulting comments, or personal/political attacks.
- Repeated disruptive behavior in issues, pull requests, or community spaces.

### Enforcement Responsibilities
Maintainers may remove, edit, or reject comments, commits, code, wiki edits, issues, and other contributions that are not aligned with this Code of Conduct, or temporarily/permanently ban contributors for behaviors that they deem inappropriate, threatening, offensive, or harmful.

### Reporting
If you experience or witness unacceptable behavior:
1. Prefer private contact through the GitHub Security Advisory form or by sending a direct message to the maintainers via GitHub (see `SECURITY.md`).
2. If privacy is not an issue, open an Issue and clearly mark it as a Code of Conduct concern.
3. Provide as much context as possible so we can investigate quickly.

### Enforcement Guidelines
Maintainers will follow these guidelines:
1. **Correction** – Written warning, clarification of expectations.
2. **Warning** – Consequences for continued behavior.
3. **Temporary Ban** – Removal from interactions for a limited time.
4. **Permanent Ban** – Removal from the community.

### Attribution
This Code of Conduct adapts the [Contributor Covenant](https://www.contributor-covenant.org/) v2.1 with bilingual guidance for this project.

## Español

### Nuestro Compromiso
Queremos que Syncopation sea una comunidad acogedora. Quienes contribuyen, mantienen o juegan se comprometen a crear un espacio libre de acoso sin importar edad, complexión, discapacidad, etnia, identidad o expresión de género, experiencia, nacionalidad, apariencia personal, raza, religión u orientación sexual.

### Nuestros Estándares
Ejemplos de comportamiento positivo:
- Mostrar empatía y respeto por distintos puntos de vista y experiencias.
- Dar y recibir retroalimentación constructiva con amabilidad.
- Pensar siempre en el beneficio de la comunidad y de la experiencia de juego.
- Usar lenguaje inclusivo en código, documentación y conversaciones.

Comportamientos inaceptables:
- Acoso, intimidación o lenguaje/discriminación ofensiva.
- Publicar información privada sin permiso explícito.
- Comentarios trolls, insultos o ataques personales/políticos.
- Conducta disruptiva repetida en issues, pull requests u otros espacios.

### Responsabilidades de Aplicación
El equipo mantenedor puede eliminar, editar o rechazar comentarios, commits, código, issues y otros aportes que no se alineen con este código, o vetar de manera temporal o permanente a quienes adopten conductas inapropiadas, amenazantes u ofensivas.

### Reportes
Si experimentas o ves una conducta inaceptable:
1. Prioriza el contacto privado mediante el formulario de GitHub Security Advisory o enviando un mensaje directo a las personas mantenedoras (ver `SECURITY.md`).
2. Si no se requiere privacidad, abre un Issue e indícalo claramente como asunto del Código de Conducta.
3. Proporciona el mayor contexto posible para facilitar la investigación.

### Guía de Aplicación
Las acciones disciplinarias siguen estas etapas:
1. **Corrección** – Advertencia escrita y aclaración de expectativas.
2. **Advertencia** – Se especifican las consecuencias si continúa el comportamiento.
3. **Suspensión temporal** – Se limita la participación por un periodo definido.
4. **Expulsión permanente** – Se elimina el acceso a la comunidad.

### Atribución
Este Código de Conducta adapta la [Contributor Covenant](https://www.contributor-covenant.org/) v2.1 con lineamientos bilingües para este proyecto.
