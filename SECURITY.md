# Security Policy / Política de Seguridad

## Supported Versions / Versiones con soporte
We currently support the latest `main` branch and the most recent public release (if tagged). Older versions might not receive fixes. / Soportamos la rama `main` y la versión pública más reciente (si existe etiqueta). Versiones antiguas podrían no recibir parches.

## Reporting a Vulnerability / Reportar una Vulnerabilidad
**English**
1. Create a private advisory at: https://github.com/Misstery13/Syncopation/security/advisories/new
2. Provide reproduction steps, impact, and suggested fixes if available.
3. Expect an acknowledgment within 3 business days and a follow-up plan within 7 business days.
4. Please do not open a public issue until we coordinate a disclosure timeline.

**Español**
1. Crea un aviso privado en: https://github.com/Misstery13/Syncopation/security/advisories/new
2. Describe pasos de reproducción, impacto y posibles soluciones.
3. Recibirás confirmación en un máximo de 3 días hábiles y un plan de seguimiento dentro de 7 días hábiles.
4. Evita abrir un Issue público hasta coordinar un calendario de divulgación.

## Preferred Communication / Comunicación Preferida
If the advisory form is unavailable, send a direct message to the maintainers through GitHub or mention in an Issue requesting a secure channel. / Si el formulario no está disponible, envía un mensaje directo a las personas mantenedoras en GitHub o menciona en un Issue que necesitas un canal seguro.

## Expectations / Expectativas
- We may request additional details or proof-of-concept snippets. / Podemos solicitar detalles extra o pruebas de concepto.
- We will credit reporters unless anonymity is requested. / Reconoceremos a quien reporte salvo que pida anonimato.
- Coordinated disclosure is appreciated so we can patch and release before public discussion. / Agradecemos la divulgación coordinada para publicar parches antes de hacerlo público.
