# Syncopation Game – Description / Descripción

**English**
Syncopation is a browser-based rhythm-fighting playground built with Phaser 3 and TypeScript. Players chain karate-inspired moves on syncopated cues, backed by original sprites, UI, and music layers that react to performance. The project targets quick iteration with Vite, custom asset tooling, and automated deployments.

**Español**
Syncopation es un juego web de ritmo y combate creado con Phaser 3 y TypeScript. Las jugadoras encadenan movimientos inspirados en karate siguiendo señales sincopadas, acompañadas de sprites, interfaz y música originales que reaccionan a su desempeño. El proyecto está optimizado para iterar rápido con Vite, herramientas propias para assets y despliegues automatizados.
